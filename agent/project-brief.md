# Project Brief

## Goal

Build a personal website for the domain `премиум-продюсер.рф`.

## Known Inputs

- Domain: `премиум-продюсер.рф`
- Owner has GitHub and wants the project maintained there.
- The project must include an `agent/` folder.
- The `agent/` folder must include `memory/` for project context and continuity.

## Open Questions

- Preferred name/brand to show on the site.
- Target audience and offer positioning.
- Required pages and sections.
- Preferred technology stack.
- GitHub repository URL or desired repository name.
- Hosting/deployment target.
