# Agent Workspace

This folder stores working documents used by the coding agent while building the personal website.

## Structure

- `memory/` - persistent project memory and context notes.
- `local/` - local-only user data and private notes; ignored by git.
- `project-brief.md` - current project brief and assumptions.
- `worklog.md` - chronological log of work performed.

## Rule

Keep decisions, assumptions, and next steps documented here so future work can continue with context.

Do not commit private user data, credentials, tokens, personal contact lists, or unpublished materials. Store those in `agent/local/`, which is intentionally ignored by git.
